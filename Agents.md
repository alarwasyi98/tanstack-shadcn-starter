# TanStack Start + shadcn/ui Starter

## Commands

```bash
pnpm dev          # Dev server (port 3000)
pnpm build        # Production build
pnpm test         # Vitest tests
pnpm lint         # ESLint
pnpm format       # Prettier (write)
pnpm typecheck    # TypeScript check
```

## Architecture

- **React 19** + **TypeScript** (strict mode) + **TanStack Router** (file-based)
- **Vite** + **Nitro** for SSR
- **Tailwind CSS v4** with OKLCH color palette
- **shadcn/ui** components (33 components pre-installed)

## Key Directories

| Path                 | Purpose                                                |
| -------------------- | ------------------------------------------------------ |
| `src/routes/`        | File-based routing (auto-generates `routeTree.gen.ts`) |
| `src/components/ui/` | shadcn/ui primitives                                   |
| `src/components/`    | Custom composed components                             |
| `src/lib/`           | Utilities: `cn()`, theme provider, `useTheme()`        |
| `src/hooks/`         | `useIsMobile()` (breakpoint: 768px)                    |

## Important Rules

- **Never edit `src/routeTree.gen.ts`** — auto-generated by TanStack Router
- Use `cn()` from `@/lib/utils` for class merging
- Use `npx shadcn@latest add <component>` to add new components
- Theme: `useTheme()` hook with `ThemeProvider` wraps app

## Design System

See `design-system.jsonc` for complete token reference:

- Colors: OKLCH format (light + dark modes)
- Typography: Geist Variable font
- Motion: custom easing (`ease-out`, `ease-spring`)
- Radius scale: `--radius` base = 0.625rem

View live docs at `/design-system` when dev server is running.

## Imports

import { Button } from "@/components/ui/button"
